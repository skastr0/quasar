# Security Policy

## Supported Versions

Quasar is in initial public development. Security fixes target the latest
released version and the default branch.

## Reporting a Vulnerability

Use GitHub private vulnerability reporting for this repository when available.
If private reporting is unavailable, open a minimal public issue asking for a
secure contact path and do not include exploit details, secrets, or private
session data.

Quasar ingests local agent histories, so reports involving data exposure,
credential handling, redaction, path traversal, or unsafe file access are
security-relevant.
